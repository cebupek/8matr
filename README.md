# 🌹 Романтический подарок на 8 Марта

Глубокий, романтичный интерактивный подарок с 8 уникальными сюрпризами, созданный специально для самой особенной женщины. Включает полноценный сервер с пингованием каждые 5 секунд.

## 💝 Восемь подарков от сердца

Каждый подарок - это не просто интерактивный элемент, а искренняя часть чувств:

### 1. 💌 Письмо о нас
Настоящее любовное письмо, написанное от души. Слова, которые трудно произнести вслух, но которые живут в сердце. Это не шаблонные фразы, а глубокое признание в том, что значит любимый человек.

**Объём:** Большое развёрнутое письмо на несколько экранов
**Стиль:** Искренний, эмоциональный, романтичный

### 2. 📖 Хроника моментов
Временная шкала ваших отношений - от первой встречи до настоящего момента. Каждый этап отношений описан с нежностью и вниманием к деталям.

**Формат:** Интерактивная timeline с 6 ключевыми моментами
**Дизайн:** Элегантная вертикальная линия времени с описаниями

### 3. 💝 Сто причин
Почему она - та самая единственная. Десять карточек с глубокими причинами, почему вы любите именно её. Каждая причина - это не поверхностный комплимент, а искреннее признание.

**Содержание:** 10 развёрнутых причин с эмоциональным текстом
**Оформление:** Красивые карточки с иконками и подробными описаниями

### 4. 💻 Код нашей любви
Уникальное задание - переписать код, который описывает вашу любовь. Код написан на JavaScript и содержит романтическую логику отношений. Если не получается написать - есть кнопка "Пропустить", которая всё равно покажет результат.

**Особенность:** Реальный работающий код с проверкой
**Результат:** Романтическое сообщение после выполнения
**Альтернатива:** Кнопка пропуска с тем же финалом

### 5. 🌹 Сад желаний
12 роз, каждая из которых скрывает особое желание для неё и ваших отношений. Нажимая на розы, они "распускаются" и показывают глубокие, искренние пожелания.

**Интерактив:** Каждая роза раскрывается с анимацией
**Пожелания:** 12 романтичных и серьёзных желаний о будущем

### 6. ✨ Созвездие надежд
Создайте вместе звёздное небо мечтаний. Каждая звезда - это надежда на будущее, мечта о совместной жизни. Кликая по тёмному небу, появляются новые звёзды.

**Механика:** Интерактивное создание звёзд кликами
**Атмосфера:** Тёмное космическое небо с мерцающими звёздами
**Смысл:** Каждая звезда сопровождается пожеланием

### 7. 🎭 Театр чувств
Пятиактная история ваших отношений, написанная как театральная пьеса. От первой встречи до осознания "навсегда". Каждый акт - это важный этап в развитии чувств.

**Структура:** 5 актов с глубоким содержанием
**Объём:** Большой текст в стиле художественного произведения
**Стиль:** Драматургический, поэтичный, глубокий

### 8. 🌸 Банка мечтаний
8 мечтаний о совместном будущем. Каждая мечта - это серьёзное желание о вашей жизни вместе: от совместных рассветов до старости рядом.

**Содержание:** 8 глубоких мечтаний о будущем
**Темы:** Дом, семья, путешествия, старость вместе, вечная любовь
**Оформление:** Красивые карточки, которые раскрываются при клике

## 🚀 Запуск проекта

### Установка и запуск

```bash

npm start
# или
node server_romantic.js
```

Затем откройте в браузере: **http://localhost:3000**

### Альтернативный запуск

Можно просто открыть файл `8march_romantic.html` в браузере - всё будет работать, но без серверных пингов и статистики подключений.

## 📡 Сервер с пингованием

Сервер отправляет пинги каждые **5 секунд** всем подключённым клиентам:

- **Технология:** Server-Sent Events (SSE)
- **Интервал:** 5000 миллисекунд (5 секунд)
- **Индикатор:** В правом верхнем углу показывается статус соединения
- **Информация:** Время последнего пинга, количество активных клиентов, время работы сервера

### Доступные эндпоинты

- `http://localhost:3000/` - главная страница с подарками
- `http://localhost:3000/ping` - Server-Sent Events для пингов
- `http://localhost:3000/stats` - JSON со статистикой (количество клиентов, время работы)
